## タイポグラフィ

### フォント

| 用途         | フォント                                         |
| ------------ | ------------------------------------------------ |
| 本文・UI全般 | Montserrat（欧文）+ Noto Sans JP（和文）         |
| フォールバック | Yu Gothic → -apple-system → Hiragino Kaku Gothic |

和欧混植構成。ブラウザは Montserrat でラテン文字を描画し、グリフのない日本語は Noto Sans JP にフォールバックする。Montserrat は `/public/font/` にセルフホスト（woff2）、Noto Sans JP は Google Fonts から読み込む。

フォント最適化として `font-feature-settings: "palt"` を本文に適用し、和文プロポーショナルメトリクスを有効化している。

### テキストサイズ

| 役割                         | 値の例                               |
| ---------------------------- | ------------------------------------ |
| ページ・セクション見出し大   | `text-[32px] sm:text-[22px]`         |
| セクション見出し             | `text-[24px] sm:text-[18px]`         |
| カード・ブロックタイトル     | `text-[20px] sm:text-[16px]`         |
| 本文                         | `text-[16px] sm:text-[14px]`         |
| サブテキスト・ラベル         | `text-[14px] sm:text-[12px]`         |
| メタデータ・タグ・日付       | `text-[12px] sm:text-[10px]`         |
| 最小補助テキスト             | `text-[10px]`                        |

### ルール

- **最小フォントサイズは `10px` とする。** それより小さいサイズは使わない
- 見出しは `font-bold`（700）または `font-black`（900）を使う
- 本文は `font-normal`（400）、強調箇所に `font-medium`（500）
- 行間（line-height）: 本文 `1.8`、見出し `1.32〜1.48`、バッジ類 `1`
- 字間（letter-spacing）: 本文 `0.06em`、見出し `0.04〜0.08em`
- `whitespace-nowrap` を使い、重要なラベルが改行されないよう保護する

## スペーシング

### 基本単位

スペーシングは **8px 基準** の倍数系。デスクトップの値の約 50% を SP に適用する。

| 用途                   | デスクトップ    | SP（sm:）       |
| ---------------------- | --------------- | --------------- |
| ページ横余白           | `px-[48px]`     | `px-[24px]`     |
| ページ縦余白（上）     | `pt-[44px]`     | `pt-[20px]`     |
| ページ縦余白（下）     | `pb-[48px]`     | `pb-[24px]`     |
| ヒーロー上余白         | `pt-[134px]`    | `pt-[80px]`     |
| セクション間           | `gap-[64px]`    | `gap-[40px]`    |
| カード内コンテンツ     | `gap-[12px]`    | `gap-[8px]`     |
| アイコンとテキスト     | `gap-[8px]`     | `gap-[6px]`     |
| 記事内 h2 上余白       | `mt-[120px]`    | `mt-[60px]`     |
| 記事内 h3 上余白       | `mt-[96px]`     | `mt-[48px]`     |
| 記事内 h4 上余白       | `mt-[72px]`     | `mt-[36px]`     |
| 記事内段落上余白       | `mt-[16px]`     | `mt-[12px]`     |

### レイアウト幅

```
max-w-[1280px] px-[48px] w-full mx-auto sm:max-w-[430px] sm:px-[24px]
```

`.inner` クラスとして共通化。コンテンツはデスクトップ最大幅 1280px、SP は最大幅 430px で中央寄せ。

## コンポーネント

### Card（`card.astro`）

ブログ記事一覧の基本コンテナ。

```astro
<!-- サムネイル・タイトル・説明・タグ・日付で構成 -->
<article class="max-w-[408px] sm:max-w-full">
  <a class="duration opacity-hover">...</a>
</article>
```

- カードの中にカードは入れない
- ホバー時は `.opacity-hover` でフェードアウト（opacity 0.48）

### Header（`header.astro`）

```astro
<header class="fixed z-50 w-full bg-[#FFF]/[0.64] backdrop-blur-lg sm:bg-[#000]/[0]">
```

- SP では背景を透明にし、メニュー展開時にオーバーレイを表示
- ロゴ: デスクトップ `w-[88px]`、SP `w-[72px]`
- ハンバーガーメニュー: `is-active` クラスでアイコン切替

### Navigation（`navigation.astro`）

カテゴリ・タグを一覧表示するドロップダウン型ナビゲーション。`.js-filter` / `.js-filter-menu` クラスで開閉制御。

### Pager（`pager.astro`）

ページネーション。現在ページは `text-[#0079C9] font-bold`、非アクティブは `text-[#333]`。

### NoteSlider（`noteSlider.astro`）

Swiper を使った note.com RSS フィードのカルーセル。

## アイコン

カスタム SVG アイコンを `icon.astro` コンポーネントで読み込む。ファイルは `/public/img/` に格納。

```astro
<Icon filename="icon_arrow_right" color="#0079C9" customClass="w-[16px] h-[16px]" />
```

| サイズ              | 用途                         |
| ------------------- | ---------------------------- |
| `w-[12px] h-[12px]` | 最小補助アイコン             |
| `w-[16px] h-[16px]` | 標準（インライン・ボタン内） |
| `w-[20px] h-[20px]` | 中型（リスト先頭等）         |
| `w-[24px] h-[24px]` | ヘッダーアイコン             |
| `w-[32px] h-[32px]` | ロゴ・大型アイコン           |

SP では対応するサイズの 50〜75% 程度に縮小する（例: `w-[32px] sm:w-[16px]`）。

## インタラクション

### ホバー

| 要素             | ホバー                                          |
| ---------------- | ----------------------------------------------- |
| カード・リンク   | `opacity-[0.48]`（`.opacity-hover` クラス）     |
| アイコンリンク   | X 軸シェイクアニメーション（`.transrate-x-animation`） |
| ナビゲーション   | `opacity-[0.48]` でフェード                     |
| ボタン           | `bg-[#0079C9]/[0.8]`（透過度下げ）             |

### トランジション

`.duration` クラスで統一（`transition-all duration-[240ms] ease-in-out`）。

```astro
<a class="duration opacity-hover">...</a>
```

### アニメーション

| 種類               | 詳細                                           |
| ------------------ | ---------------------------------------------- |
| フェード           | opacity 1 → 0.48、240ms ease-in-out            |
| シェイク           | translateX 0 → 20px → 0、480ms ease-in-out     |
| ブリンク           | スクロール誘導インジケーター、1.6s 無限ループ  |
| メニューオープン   | `hidden` / `block` クラス切替（JS）           |

ブレークポイントは `sm`（max-width: 1416px）**1種類のみ**。デスクトップファーストで記述し、`sm:` で SP・タブレットを上書きする。

| 要素                     | デスクトップ              | SP（sm:）                      |
| ------------------------ | ------------------------- | ------------------------------ |
| コンテンツ最大幅         | `max-w-[1280px] px-[48px]` | `max-w-[430px] px-[24px]`     |
| ヘッダー背景             | `bg-[#FFF]/[0.64]`        | `bg-[#000]/[0]`（透明）        |
| フォントサイズ           | 基準値                    | 約 75〜80% に縮小              |
| 余白・パディング         | 基準値                    | 約 50% に縮小                  |
| カードグリッド列数       | 複数列                    | 1列                            |
| ナビゲーション           | インライン表示            | ハンバーガーメニュー           |

## 記事コンテンツスタイリング

`.post` クラス配下のコンテンツ要素に専用スタイルを適用。

| 要素         | スタイル概要                                              |
| ------------ | --------------------------------------------------------- |
| h2           | 大見出し、上 120px（SP: 60px）のマージン                  |
| h3           | 中見出し、上 96px（SP: 48px）のマージン                   |
| h4           | 小見出し、上 72px（SP: 36px）のマージン                   |

| table        | SP で横スクロール可能なラッパー内に配置                   |

## アクセシビリティ

- リンクテキストは役割を明示し、「こちら」のみの anchor を使わない
- 画像には必ず `alt` を付ける（装飾のみの場合は `alt=""`）
- フォーカスリングは `focusRectangle.svg` を使った視覚的フォーカスインジケーターで補完
- コントラスト比は WCAG 2.1 AA 準拠を最低基準とする

| 基準                   | 通常テキスト | 大きなテキスト（18pt/24px 以上） |
| ---------------------- | ------------ | -------------------------------- |
| レベル AA（最低基準）  | 4.5:1 以上   | 3:1 以上                         |
| レベル AAA（推奨基準） | 7:1 以上     | 4.5:1 以上                       |

- 本プロジェクトのメインカラー `#0079C9` on `#FFF` のコントラスト比は約 4.6:1（AA 適合）
- 本文 `#333` on `#FFF` は約 12.6:1（AAA 適合）
- 補助テキスト `#999` on `#FFF` は約 2.8:1 — **AA 非適合のため `#666` 以上を使用すること**
